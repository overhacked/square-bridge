To Do list (TODO)

Import and export files can contain all the information on your To Do list. The keyword for the To Do list is TODO.

Column headings

The following table shows the column headings for the To Do list.

REFNUM

(Export files only) A unique number that identifies an entry in the list.

ISDONE

Indicates whether the task is complete. If you are creating an import file, enter one of these keywords in the ISDONE column:

 

Y

Yes.

 

N

No.

DATE

The date you want QuickBooks to place the task on your Reminders list.

DESC

Your notes about the task.
